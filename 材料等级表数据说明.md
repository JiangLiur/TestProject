# 68材料等级表结构化数据

本项目从《68材料等级表.pdf》中提取了管道材料等级信息，并转换为结构化数据格式。

## 数据文件

- **材料等级表数据.csv** - CSV格式，适合Excel等电子表格软件打开
- **材料等级表数据.json** - JSON格式，适合程序化处理
- **extract_material_data.py** - 数据提取脚本

## 数据字段说明

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 等级名称 | 管道等级代号 | C1L01, C1C01, C9C01 |
| 公称直径DN | 公称直径（毫米） | 15, 20, 25, 40... |
| 材料 | 管道材料牌号 | Q345E, L245N, 06Cr19Ni10 |
| 外径φ | 管道外径（毫米） | 21, 27, 34... |
| 壁厚mm | 管道壁厚（毫米） | 3, 4, 5... |
| 标准号 | 执行标准 | GB/T6479, GB/T9711 |
| 压力等级 | 法兰压力等级 | Class150, Class900 |
| 设计压力 | 管道设计压力 | 1.6MPa, 13.9MPa |
| 设计温度 | 管道设计温度范围 | -40℃—100℃ |

## 数据使用示例

### Python读取JSON数据

```python
import json

with open('材料等级表数据.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# 筛选特定等级
c1l01_data = [item for item in data if item['等级名称'] == 'C1L01']

# 按公称直径查询
dn50_data = [item for item in data if item['公称直径DN'] == 50]
```

### Excel/CSV使用

直接在Excel中打开 `材料等级表数据.csv` 文件即可进行筛选、排序、统计等操作。
